Installation of cocotb
======================

.. versionadded:: 1.2

The **latest release** of cocotb can be installed by running

.. code-block:: bash

    pip install --global-option build_ext --global-option --compiler=mingw32 cocotb

.. include:: install_pip3_warning.rst

.. include:: install_user_local.rst

If you want to install the **development version** of cocotb, :ref:`instructions are here<install-devel-windows>`.
